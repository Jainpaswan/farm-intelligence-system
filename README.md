## ✨ **Overview**

The **Farm Intelligence System** is a full-stack AI solution designed to support farmers, agronomists, and researchers.
It combines **Machine Learning**, **Geolocation**, and a **modern web UI** to deliver:

✔ Crop Recommendation
✔ Yield Prediction
✔ Auto Location Detection
✔ Real-time ML API Integration
✔ Clean & Responsive UI
✔ Notebook-based Model Training

---

## 🧠 **Key Features**

### 🌱 **AI Crop Recommendation**

Predicts the best crop based on soil nutrients (N, P, K), pH, rainfall, humidity, temperature & region.

### 📈 **Yield Prediction**

Accurately estimates crop output (tons/hectare) using trained ML models.

### 📍 **Auto Location Detection**

Frontend automatically detects the user’s location to improve crop accuracy.

### 🖥️ **Modern Frontend**

* Built using React/Next.js
* Clean UI + Loader + Error Handling
* Real-time model results

### 🔌 **Machine Learning API**

* Model Loader V2
* Fast & optimized inference
* Separate ML pipeline folder

## 🛠️ **Local Setup**

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Jainpaswan/farm-intelligence-system
cd farm-intelligence-system
```

---

### 2️⃣ Start Frontend

```bash
cd frontend
npm install
npm run dev
```

---

### 3️⃣ Start ML API

```bash
cd "MI Model API"
pip install -r requirements.txt
python app.py
```
